# Requirements

[Node with npm](https://nodejs.org/)

# Installation

1. Goto `client` folder and run:
```
> npm install
```
2. Go back to main folder and run same command.

# Build and usage

1. From main folder run: 
```
> npm run build
```
2. Start server:
```
> npm run server:prod
```
3. Go to `localhost:3001` in your browser.

# Development

```
> npm start
```
